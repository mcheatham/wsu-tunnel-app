distanceTo(that) → {number}

Computes the distance between this coordinate points and another using the Haversine formula.
NOTE: This algorithm has been tweaked for the local radius of the Earth at Wright State University (39.7846° N, 84.0583° W). For use at another location, modify the radius variable.
