Class: Graph

Graph(numNodes, numEdges, nodeLoaderopt, edgeLoaderopt)

Class representing data about the entire map graph

Constructor

new Graph(numNodes, numEdges, nodeLoaderopt, edgeLoaderopt)

Creates a new graph object with a pre-specified number of nodes and edges. The functions used to load the data for the nodes and edges can be specified as optional parameters. If no value is specified or null are specified, the respective default functions will be used
Parameters:

numNodes number The number of nodes in the graph
numEdges number The number of edges in the graph
nodeLoader function <optional>
 null The function to load the node data with. If null is specified, the default node laoding function will be used
edgeLoader function <optional>
 null The function to load the edge data with. If null is specified, the default edge loading function will be used.
